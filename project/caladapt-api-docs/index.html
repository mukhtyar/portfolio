Background

Cal-Adapt provides peer-reviewed climate data and climate tools for California. The Cal-Adapt API provides programmatic access to climate data hosted on Cal-Adapt. It is built using Django, Django REST framework, and Django-Spillway, an open source library developed at the GIF.

My Work

I set up the documentation stack for the Cal-Adapt API using Sphinx, a popular documentation generator for Python projects, using a theme from Read the Docs. I wrote tutorials and code samples to help users interested in working with the Cal-Adapt API.
